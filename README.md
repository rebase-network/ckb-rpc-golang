# Ckb RPC Tool

## RPC

```
export API_URL=http://1.0.0.1:8114
./ckb-rpc-mac -list

./ckb-rpc-mac -rpc gbt
./ckb-rpc-mac -rpc tipheader
./ckb-rpc-mac -rpc tipblocknumber
./ckb-rpc-mac -rpc currentepoch
./ckb-rpc-mac -rpc peers
./ckb-rpc-mac -rpc localnodeinfo
./ckb-rpc-mac -rpc txpoolinfo
```
